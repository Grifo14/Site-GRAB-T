# Website GRAB!T 🍔🛵

## Sobre o Website

O website da GRAB!T foi desenvolvido com o objetivo de apresentar o projeto e dar a conhecer as vantagens da nova aplicação móvel de entrega de comida rápida.
Através de uma interface apelativa e moderna, o website permite aos visitantes compreender o funcionamento da plataforma, explorar os pilares do projeto (Visão, Missão e Valores) e conhecer a equipa responsável pelo seu desenvolvimento.
Para além da apresentação institucional, a página inicial integra um vídeo demonstrativo (screencast) em formato mobile, permitindo visualizar a fluidez da aplicação em tempo real[cite: 1].

---

## Funcionalidades

O website está dividido em diferentes páginas e inclui as seguintes secções:
- Página inicial (Início) com apresentação direta da proposta de valor da GRAB!T e atalhos para descarregar a app[cite: 1];
- Secção explicativa de "Como funciona", detalhada em 3 passos simples (Escolhe, Pede, Desfruta)[cite: 1];
- Destaque para as principais vantagens da plataforma, como a rapidez de entrega, opções variadas e pagamentos seguros[cite: 1];
- Página "Sobre" dedicada aos compromissos da marca, focando-se na sustentabilidade ecológica, comércio local e excelência;
- Apresentação detalhada da equipa de desenvolvimento (os fundadores), com os respetivos cargos e responsabilidades;
- Sistema de navegação responsivo com botão dinâmico de retorno ao topo (Scroll to Top).

---

## Tecnologias Utilizadas

O desenvolvimento do website recorreu às tecnologias web standard:
- **HTML5** (Estruturação semântica do conteúdo e páginas múltiples)
- **CSS3** (Estilização avançada com CSS Variables, Flexbox, Grid e animações `@keyframes`)[cite: 3]
- **JavaScript** (Lógica de interação frontend para o botão de scroll dinâmico)[cite: 1]
